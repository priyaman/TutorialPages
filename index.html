Windows 7 Spark Tutorial
Hello World Program

Though we can run Spark in stand alone mode and give it files as input. To make a reliable data source, a HDFS(Hadoop Distributed File System) is required. As Spark supports loading data from a HDFS, we would need an installation of a HDFS before we start with an example.

Step 1: Installing Hadoop

We can either download the source code for Hadoop and build it. You would need to setup Maven or some other build tool to build. You are welcome to do that but since this is a quick and snappy hello world project, I suggest that you download the binaries.

Go to this link and download the tar.gz : http://www.apache.org/dyn/closer.cgi/hadoop/common/hadoop-2.5.2/hadoop-2.5.2.tar.gz

Extract it. Since you are on Windows and tar.gz is not a commonly used compression technique, you would probably need to download WinRAR to unzip it.

Step 2: Install Spark

Go to the Spark download page at https://spark.apache.org/downloads.html, choose a version of Spark and download the pre-built binary for hadoop.

LINK: http://www.apache.org/dyn/closer.cgi/spark/spark-1.0.2/spark-1.0.2-bin-hadoop2.tgz

Again, you are welcome to download the source code and build it, but it is a big pain in the ass to build spark because of the number of dependencies
